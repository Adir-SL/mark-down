# Ocean Theme A dark theme built around **teal and aqua accents** on a deep navy canvas — like staring into the deep sea. Visit the [mark-down.js repo](https://github.com/Adir-SL/mark-down) to get started. > [!TIP] > The `teal` palette reads beautifully at night — easy on the eyes for long sessions. > [!WARNING] > Combining multiple theme files on one page requires scoped CSS selectors. ## Code example ```js function greet(name) { return `Hello, ${name}!`; } console.log(greet("world")); ``` ## What's included - Deep navy `#0a1628` background — deeper than dark - `#2dd4bf` teal link color that shimmers - Coral warnings, sky-blue notes, indigo callouts | Variable | Value | | -------- | ----- | | `--md-bg` | `#0a1628` | | `--md-color` | `#cdd9e5` | | `--md-link-color` | `#2dd4bf` |
