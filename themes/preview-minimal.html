# Minimal Theme Stripped back to the *essentials* — **no border-radius**, no colored backgrounds, no decoration. Just pure prose. Visit the [mark-down.js repo](https://github.com/Adir-SL/mark-down) to get started. > [!NOTE] > This theme inherits your page's `color` and `font-size` — it adapts to whatever context it's dropped into. > [!TIP] > Combine with a custom `font-family` on your page body to instantly style all `mark-down` elements. ## Code example ```js function greet(name) { return `Hello, ${name}!`; } console.log(greet("world")); ``` ## What's included - Transparent code backgrounds — no visual noise - Zero border-radius on all elements - Georgia serif with a `1.85` line height | Variable | Value | | -------- | ----- | | `--md-bg` | `#ffffff` | | `--md-color` | `inherit` | | `--md-link-color` | `inherit` |
