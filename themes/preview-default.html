# Default The built-in style — clean, neutral, and inherits your page's colors. No theme file needed. Visit the [mark-down.js repo](https://github.com/Adir-SL/mark-down) to get started. > [!NOTE] > This is what you get out of the box — no theme CSS loaded. > [!TIP] > The default adapts to your page's `color` and `font-size` automatically. ## Code example ```js function greet(name) { return `Hello, ${name}!`; } console.log(greet("world")); ``` ## What's included - Inherits `currentColor` from the page - Sans-serif system font stack - Subtle grey code backgrounds and rounded borders | Variable | Default | | -------- | ------- | | `--md-bg` | `transparent` | | `--md-color` | `inherit` | | `--md-link-color` | `#0969da` |
