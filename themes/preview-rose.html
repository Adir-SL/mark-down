# Rose Theme An *elegant, literary* theme with **deep rose accents** on a soft blush canvas — timeless and refined. Visit the [mark-down.js repo](https://github.com/Adir-SL/mark-down) to get started. > [!NOTE] > The Georgia serif font gives your writing an *editorial* quality — perfect for blogs and personal sites. > [!IMPORTANT] > Callout colors are rich and purposeful — violet for important, sage for tips, rose for caution. ## Code example ```js function greet(name) { return `Hello, ${name}!`; } console.log(greet("world")); ``` ## What's included - Soft blush `#fdf2f4` background — warm without being yellow - Georgia serif at `1.85` line height for comfortable reading - Deep rose `#b5174e` links and accents | Variable | Value | | -------- | ----- | | `--md-bg` | `#fdf2f4` | | `--md-color` | `#2d1b22` | | `--md-link-color` | `#b5174e` |
